# Jogo sobre pedra, papel e tesoura.

Alunos: Alex José das Neves Junior, Hariel Mohamed, Kaio Richard, Tomas Ernestro.

Justificativa: 

O desenvolvimento do código, justifica-se pela necessidade de um jogo multiplayer de pedra, papel e tesoura entre duas pessoas rodando o mesmo código em máquinas diferentes, estando conectadas no mesmo servidor.

Como rodar:

Linguagem utilizada: Python, IDE utilizado: Visual Studio Code, servidor local (redis 2.6): https://github.com/MSOpenTech/redis/raw/2.6/bin/release/redisbin64.zip

Explicação: 

Um jogo multiplayer de pedra, papel e tesoura, na qual, duas pessoas jogam (ambos rodando o mesmo código), onde, uma cria uma sala, e a outra entra na sala, se conectando por via servidor (cmd: venv\Scripts\Python.exe main.py), e jogando pedra, papel e tesoura.
